Documentation

Thank you for purchasing Elite Pro. This documentation covers the installation and configuration of the theme to help you create a world-class luxury store.

1. Installation

Follow these steps to upload the theme to Shopify:

  1. Download the package from ThemeForest and unzip it.
  2. Find elite-pro-theme.zip inside the folder.
  3. In Shopify Admin, go to Online Store > Themes.
  4. Click Add Theme > Upload zip file.
  5. Once uploaded, click Actions > Publish.

2. Key Sections

Cinematic Hero

To set up the Hero video, go to the Theme Editor and upload your MP4 file. For best results, use a 16:9 aspect ratio.

Section > Cinematic Hero > Video URL / Upload

Related Products

This section uses Shopify AI. To enable it, ensure your products have Collections assigned.

3. Recommended Image Sizes

To maintain the "Cinematic" look, use these dimensions:

  • Product Images: 1000 x 1250px (Portrait)
  • Hero Background: 1920 x 1080px
  • Collection Thumbnails: 800 x 800px
