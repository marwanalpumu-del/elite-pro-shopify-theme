### Design Tokens (CSS Variables)
The theme uses a centralized variable system in `:root` to ensure consistency:
- `--elite-gold-primary`: `#D4AF37`
- `--bg-color`: `#050505`
- `--transition-apple`: `cubic-bezier(0.16, 1, 0.3, 1)`
